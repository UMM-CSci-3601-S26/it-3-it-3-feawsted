Inventory

Welcome to the inventory page! Here, you can view all items currently in the inventory, as well as filter and sort them by various attributes. You can also edit or delete existing inventory items, or add new ones using the "Add Inventory" button on the bottom right. To edit an item, click the three dots on the right side of the item row.

Use the fields below to filter the inventory by item, brand, color, size, type, and material. The filters will be applied as you type, and you can clear them all at once using the "Clear" button on the bottom. You can also sort the inventory by any column by clicking on the column header.
